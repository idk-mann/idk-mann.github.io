My Projects

- BlueSkies

BlueSkies is a solo-developed modern Android weather app designed to keep users informed without needing to constantly check the forecast. It features real-time weather updates and personalized alerts, with a clean, responsive UI built using the latest Android design principles.

Key Features:

  • Real-time weather data powered by Tomorrow.io API
  • Customizable forecast range: 1 to 7 days
  • Up to 10 personalized daily weather notifications with user-defined times
  • Map-based location picker and city autocomplete with offline support via Room
  • Dynamic background themes based on time of day at the selected location (sunrise, sunset, night, etc.)
  • Fully responsive UI with Material 3 and system-wide dark/light mode support

Technical Highlights:

  • Asynchronous loading of weather data and UI components using Kotlin Coroutines
  • Weather API integration using Retrofit for efficient, scalable network communication
  • City autocomplete via local Room database for fast, offline-capable suggestions
  • WorkManager for reliable background delivery of weather notifications
  • RecyclerView for efficient rendering of dynamic forecast lists
  • Dynamic theming tied to location-based time data (e.g. day/night transitions)
  • MVVM architecture to ensure maintainability and scalability

What I’ve Learned:

BlueSkies helped me strengthen my understanding of asynchronous programming, clean architecture, REST API integration, and advanced UI design in Android. It was a valuable learning experience, particularly in applying MVVM architecture, Kotlin Coroutines, and WorkManager for reliable background processing. I gained valuable backend experience by building features like personalized notifications and time-aware theming, which allowed me to deliver a more dynamic and user-focused experience.

- Crypto

CRYPTO is a flask application that I made to learn about JavaScript, Bootstrap and handling APIs response using restful CoinMarketCap basic plan API. CRYPTO has a main table to display all the relevant data about the crypto currencies, when a row is clicked a more detailed page is shown. A treemap made with D3.js offers data visualization of the cryptocurrency.

Due to the constraints of the free plan, this project has limited API enpoints to work with. Unfortunately, historical data endpoints are not accessible, which means the ability to generate price graphs using API calls is not feasible. Sort functions done thru the API parameters are questionable at best.
