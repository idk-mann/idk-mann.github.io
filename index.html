- Pixel Battleship

Pixel Battleship is a recreation of the classic Battleship game using JavaScript and WebSocket, albeit with a pixified look that adds a retro, pixel art style to the gameplay. I've learned a lot, especially how to structure and organize the code. There's also logic that allows the computer to search for the ship if a hit is detected. There's additional CSS and JavaScript animations for the ship waves and hit effect.

- Crypto

CRYPTO is a flask application that I made to learn about JavaScript, Bootstrap and handling APIs response using restful CoinMarketCap basic plan API. CRYPTO has a main table to display all the relevant data about the crypto currencies, when a row is clicked a more detailed page is shown. A treemap made with D3.js offers data visualization of the cryptocurrency.

Due to the constraints of the free plan, this project has limited API enpoints to work with. Unfortunately, historical data endpoints are not accessible, which means the ability to generate price graphs using API calls is not feasible. Sort functions done thru the API parameters are questionable at best.

- Snake Game

I developed a snake game that blends a modern with a nostalgic retro feel, offering a cozy and challenging experience. Built using the SDL2 library in C, the game ditches the traditional tile-based layout for a smoother design. I added random obstacles generation to increase difficulty, and introduced exciting features like food buffs that grant extra lives, adding a strategic element to the gameplay.
